Super Mario Bros.

Android Emulator Edition

Working with another member, Jayrell Quiambao and I created an Android Studio Emulator version of the beloved Super Mario Bros game in Java. Jayrell mainly coded the back-end functionality of this project while I developed the front-end aspect. I was able to get acquianted with object oriented programming and applied new elements to the user interface such as animation of Mario characters and the real time movement of the player in conjunction with user input.
