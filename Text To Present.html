Text To Present

Hackathon 2020 Project

Text to Present is a Python developed, client-to-server application that reads a text sent from a user that is displayyed onto a GUI. This program was developed for the Hackathon at UCI in 2020 by myself, Roy, and Shaka. Together, our team thought of using the Twilio API to communicate with the database. From there, the database stores all the messages sent from their respective locations within the spreadsheet and the server is able to read through this database to display it on the interface as seen below. Openpxyl and Tkinter were Python modules/frameworks used to accomplish the aforementioned functions.
